                                    # 1、标题

                                    标题：一级标题用“#+空格+文字”的格式，二级标题用“##+空格+文字”的格式，以此类推。

                                    # 1.1 新增（vuepress中可用）

type可以换成tip、warning、danger  details

                                    • 粗体 粗体的格式是：前后各两个星号 **测试** ，效果为：测试
                                    • 斜体 斜体的格式是：前后各一个星号 *测试*，效果为：测试
                                    • 分隔符 分隔符的格式是：三个星号 *** ,或者连续三个短横线---，效果为：

                                    # 3、换行

                                    1. 段内换行 段内换行使用“结尾空两个空格再回车”的方式。
                                    2. 段落换行 向下空一行
                                    3. 空多行 使用HTML的<br>标签来实现
                                    4. 附加说明 在使用typora软件时，三个换行表示真正的换行；"四个空格"功能未知。

                                    # 4、列表

                                    # 6、代码

                                    1. 段落文字中的代码 文字中的代码，用反引号printf("hello,world!");包裹。
                                    2. 独立代码块 独立的代码块用三个反引号包含，并指明类型，如：
                                    printf("hello,world!");

                                    # 7、链接

                                    1. 方式1 []和（） 格式为：[链接名](链接地址) 圆括号中可以为网络地址，也可以为本地的绝对地址、相对地址。 例如：VSCode背景图 ， 可以添加链接的标题：[VSCode背景图](C:\backgroud.jpg "链接的标题")
                                    2. 方式2 采用类似参考文献的格式 类似于参考文献，[链接名][1] ,然后在文件内容的最后面，写上[1]:链接地址

                                    # 8、图片

                                    图片的插入和链接类似，可以网络，也可以本地。 格式为：![图片名](图片的地址) 例如：![VSCode背景图](C:\backgroud.jpg) 再比如：

                                    # 9、表格

                                    表格用“|”来分隔。  ：

                                    |表头1|表头2|
|---|:----:|
|单元格1|单元格2|

                                    效果如下所示：

                                    表头1表头2
                                    单元格1单元格2

                                    注意：（空一行表示表格结束）短横线做分割线，至少要三个。 说明：用冒号:来设置左右对齐方式，如左对齐：":单元格" ；右对齐 “单元格:”；居中： “:单元格:”。

                                    (6)、表格 ctrl+t

                                      ：1) 使用ctrl+t弹出表格创建提示框； 2) 使用以下语法来创建表格：

                                    |英文代码|对应颜色|----|英文代码|对应颜色|
|:-----|:-----:|:--:|:----:|:-----:|  //表头和内容的分割符号

//其中：|:--:|表示居中对齐，|:--|表示左对齐

|red|红色|----|purple|紫色|
|green|绿色|----|skyblue|天蓝色|
|blue|蓝色|----|lightgreen|淡绿色|
|Gold|金色|----|grey|灰色|
|Orange|橙色|----|blueviolet|深紫罗兰色|

                                    效果为：

                                    英文代码对应颜色----英文代码对应颜色
                                    red红色----purple紫色
                                    green绿色----skyblue天蓝色
                                    blue蓝色----lightgreen淡绿色
                                    Gold金色----grey灰色
                                    Orange橙色----blueviolet深紫罗兰色

                                    (11)、链接和脚注

                                    链接在前面已经说过了，这里简单再提一下：[链接名](链接地址)

                                      ：脚注的格式是：

                                    在待脚注的位置写上A`[^1]`。是多少

然后在后文，空一行，再加上：B`[^1]:脚注的内容`

这部分脚注B的内容会被移动到文章的最末尾显示，并且会在A处留校一个可以跳转的链接，同时B处也有往回跳转的链接。主要注意的是，上述A和B仅是为了方便说明，实际使用是不要加上这两个字符。
