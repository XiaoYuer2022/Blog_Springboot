                                                                      # 3、print格式化打印

                                                                      基础：print(内容1，内容2，内容3，。。。)，示例：print("变量a的结果是：",a,"\n") 进阶版：使用%来进行格式化打印。 ①整数输出

                                                                      %o   ----   oct八进制
%d   ----   dec十进制
%x   ----   hex十六进制
示例：
>>> print('%o' % 20)    ,输出 24
>>> print('%d' % 20)    ,输出 20
>>> print('%x' % 20)    ,输出 14

                                                                      ②浮点数输出

                                                                      %.nf   ----   保留小数点后n位小数
%.ne   ----   保留小数点后n为小数，以指数形式输出打印
%.ng   ----   保留n位有效数字
示例：（注意：计算有效位时，小数点也算一个）
>>> print('%f' % 1.11)    ,输出 1.110000
>>> print('%.1f' % 1.11)    ,输出 1.1
>>> print('%e' % 1.11)    ,输出 1.110000e+00
>>> print('%e' % 1.11)    ,输出 1.110000e+00
>>> print('%.1e' % 1.11)    ,输出 1.1e+00
>>> print('%g' % 221.1111)    ,输出 221.111  ，默认g是6位有效数字
>>> print('%.2g' % 221.1111)    ,输出 2.2e+03 ，自动转换为科学计数法

                                                                      ③整数输出

                                                                      %0nd   ----   输出指定位数的十进制，如有效数字位数不够，前方补零
>>> print('%05n' % 123)   ,输出00123

                                                                      ④字符串输出

                                                                      %ns   ----   右对齐，占位符10位
%-ns   ----  左对齐，占位符10位
%.ns   ----  左对齐，占位符10位
%n.ms   ---- n位占位符，截取m位字符串
示例：
>>> print('%s' % 'hello world')      ,输出 hello world
>>> print('%20s' % 'hello world')    ,输出           hello word (右对齐，字符串为10，所以前面要补10个空格)
>>> print('%-20s' % 'hello world')    ,输出 hello world         (左对齐，后面补10个空格)
>>> print('%.2s' % 'hello world')    ,输出 he
>>> print('%10.2s' % 'hello world')    ,输出         he (默认右对齐，总长度为10，取两个字符，所以前面有8个空格)
>>> print('%-10.2' % 'hello world')    ,输出 he        (左对齐，输出一个he加上8个空格)

                                                                      ：函数调用时通过“键=值”形式传递参数.: 可以让函数更加清晰、容易使用，同时也清除了参数的顺序需求.：函数调用时，如果有位置参数时，位置参数必须在关键字参数的前面，但关键字参数之间不存在先后顺序

#函数调用

                                                                      # 3、匿名函数

                                                                      1、函数作为参数传递

                                                                      函数本身是可以作为参数，传入另一个函数中进行使用的。

                                                                      将函数传入的作用在于：传入计算逻辑，而非传入数据。示例如下：

                                                                      def test_func(a,b,func)：
	result=func(a,b)
	print(result)

def func1(x,y)：
	return x+y
def func2(x,y)：
	return x*y
	
# 函数调用测试
test_func(1,2,func1) # 打印结果为3
test_func(1,2,func2) # 打印结果为2

                                                                      2、lambda匿名函数

                                                                      def定义的函数都是有名称的，并且可以基于名称多次使用该函数。而lambda函数可以定义没有函数名的函数，且匿名函数仅能使用一次。

                                                                      注意：

                                                                      • 匿名函数用于临时构建一个函数，只用一次的场景
                                                                      • 匿名函数的定义中，函数体只能写一行代码，如果函数体要写多行代码，不可使用匿名函数。

                                                                      匿名函数定义的语法： lambda 传入参数：函数体 # 注意：函数体仅允许只有一行代码

                                                                      • lambda 是关键字，表示定义匿名函数
                                                                      • 传入参数表示匿名函数的形式参数，如：x, y 表示接收2个形式参数
                                                                      • 函数体，就是函数的执行逻辑，要注意：只能写一行，无法写多行代码 示例：
                                                                      def test_func(a,b,func)：
	result=func(a,b)
	print(result)

# 函数调用
test_func(1,2,lambda x,y: x+y)  # 打印输出3

my_list:list[int,str,float]=[1,"2",3.12]

                                                                      # 16.4.3 Union类型的注解

                                                                      在含有多种数据类型的数据容器中进行类型注释，就需要一点技巧了。Union类型就是解决这一问题的方法之一。

                                                                      先导包：from typing import Union 对于单一数据类型的容器或函数形参/返回值注解，可以直接使用：

                                                                      my_list:list[int]=[1,2,3]
my_dict:dict[str,int]={"HM":11,"ds":22}

                                                                      对于含有多种数据类型的容器或函数形参、返回值，就需要Union类型的帮助了。

                                                                      # 也可以将复合的这几种数据类型拆开，不过这种写法只在python3.9后才支持
# my_list:list[int,str]=[1,2,"HM","lalala"]

from typing import Union
my_list:list[Union[str,int]]=[1,2,"HM","lalala"]
my_dict:dict[str,Union[str,int]]={"HM":11,"sanaj":"heihie"}
# 使用Union[类型,类型,...]，
# 可以实现联合类型注解  在函数形参和返回值上，也是可以用的：
def func(data:Union[int,str]) -> Union[int,str]:
	pass
